Linux学习-远程登录Linux系统

2021/01/28linux点击

1.前提说明

远程访问 Linux 系统，有一个前提：Linux 启用了 SSHD 服务，该服务会监听 22 号端口。

使用ifconfig命令查看 Linux 虚拟机的 ip：

2.Xshell

  • Xshell 是目前最好的远程登录到 Linux 操作的软件，流畅的速度并且完美解决了中文乱码的问题， 是目前程序员首选的软件。
  • Xshell 是一个强大的安全终端模拟软件，它支持 SSH1, SSH2, 以及 Microsoft Windows 平台的 TELNET 协议。
  • Xshell 可以在 Windows 界面下用来访问远端不同系统下的服务器，从而比较好的达到远程控制终端的目的。

（1）文件 -> 新建，打开新建会话，输入远程 ip

（2）选择“接受并保存”

（3）输入用户名

（4）输入密码

（5）远程登录成功

3.Xftp

  • Xftp 是一个基于 windows 平台的功能强大的 SFTP、FTP 文件传输软件。使用了 Xftp 以后，windows用户能安全地在 UNIX/Linux 和 Windows PC 之间传输文件。

（1）文件 -> 新建，打开新建会话，输入远程 ip，选择 SFTP 协议，输入用户名 root

（2）输入密码

（3）连接成功

如果出现乱码问题，可进行如下设置：

(2).点击“选项”，选择 UTF-8 编码。然后回到 Xftp 右侧 Linux 文件列表，右键刷新一下即可。
